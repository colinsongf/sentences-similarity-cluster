# similarity-cluster
sim_cluster.py clusters(hierarchical clustering) inputs from file data by Levenshtein distance and displays the result with dendrogram.  
## Usage
1. prepare your data  
something like this  
'''csv:dummydata.csv
A,helloworld
B,hallawerld
C,helldwoody
D,hallowarld
E,galloworld
F,herroworld
'''
2. execute  
'''
python sim_cluster.py
'''
